Überblick

Sildesk ist eine Browser-Erweiterung für den neuen Tab. Diese Erklärung beschreibt transparent, welche Daten gespeichert werden und welche externen Dienste bei aktivierten Funktionen kontaktiert werden.

Lokale Datenspeicherung

  • Alle Dashboard-Daten (Karten, Boards, Design-Einstellungen) werden ausschließlich lokal in browser.storage.local gespeichert.
  • Es werden keine Daten an den Entwickler übermittelt.
  • Es gibt keine Analyse, kein Tracking und keine Telemetrie.

Externe Dienste

Die folgenden externen Dienste werden nur dann kontaktiert, wenn die jeweilige Funktion aktiv genutzt wird. Alle Anfragen werden direkt von deinem Browser gestellt — kein Proxy, kein Umweg über Server des Entwicklers.

Optionale Synchronisation (GitHub Gist)

Wenn du GitHub Gist Sync aktivierst, wird deine Konfiguration in einem von dir erstellten privaten GitHub Gist gespeichert. Dein persönlicher GitHub-Token wird ausschließlich lokal in browser.storage.local gespeichert und niemals an Dritte übermittelt.

Berechtigungen

  • storage / unlimitedStorage — Lokales Speichern der Konfiguration
  • tabs — Browser-Verlauf-Widget und Board-Wechsel
  • history — Anzeige zuletzt besuchter Seiten im History-Widget
  • alarms — Automatische RSS-Aktualisierung alle 30 Minuten
  • dns — Ermittlung der lokalen IP-Adresse über DNS-Auflösung
  • <all_urls> — Notwendig für externe API-Anfragen aus dem Background-Script (Speedtest, IP-Abruf, Wetter etc.) ohne CORS-Einschränkungen
